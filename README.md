# CS348Project
CS348 Project - Student Degree Planner

# AI Usage
I used AI on the project as a technical assistant that supported normalization of data and feature specifications needed for the UI. The AI tools I used were ChatGPT and Microsoft Copilot. The data I used for this project is from Purdue’s course catalog. As there is no table that has the information on courses I needed, I had to provide the catalog link to AI tools and ask them to normalize the data. Specifically, I mentioned that I need columns such as the subject code, course number, title, and number of credits. It then provided the data in a tabular format, and I would ask it to provide them as SQL insert statements to accelerate the data collection process.

Another use case was for the front-end programming using JavaScript and React. As I was not familiar with front-end UI development, using AI helped me to understand the various functions and features of JavaScript and React I could use for the project. Using AI tools also helped me to learn about UI design through the implementation of buttons, various tabs and the dropdown menus.

I verified the data provided by AI by doing a manual check on the course information. As for the front-end development, I had asked it to provide sample logic for implementing the various UI features mentioned above. I then modified the base structure provided by AI to incorporate the features needed such as dropdown menus and report filtering criteria. 

The assistance of AI tools in these areas helped me to accelerate the development of my project.
